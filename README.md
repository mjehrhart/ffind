# ffind
Fast Find is a cli written in rust using rayon parrellism for super fast results.

## Install ffind

```
brew tap mjehrhart/ff
install mjehrhart/ff/ff
```

## Update/Upgrade
```
brew reinstall mjehrhart/ff/ff
brew upgrade mjehrhart/ff/ff
```
